# 各种语言如何输出hello，world
## 1.go语言
```go
package main
import "fmt"
func main(){
    fmt.Println("hello,world")
}
```
## 2.c语言
```c
#include<stdio.h>
int main(void){
    printf("hello,world\n");
    return 0;
}
```
## 3.c++
```cpp
include<iostream>
int main(){
    cout<<"hello,world<<endl;
    return 0;
}
```
## 4.java
```java
public class hello {
    public static void main(String[] args) {
        System.out.println("hello,world");
    }
}
```
## 5.js
```js
console.log('hello,world');
```
## 6.python
```py
print("hello,world")
```
